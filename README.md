# Module 3 – PL/SQL Programming

This repository contains learning materials, notes, and practical implementations for **PL/SQL Programming**, covering the core concepts required for developing efficient database applications in Oracle Database environments.

## Overview

PL/SQL (Procedural Language/SQL) is Oracle's procedural extension to SQL that enables developers to combine SQL statements with programming constructs such as variables, loops, conditions, cursors, exception handling, procedures, functions, packages, and triggers. This module focuses on building database-driven applications, automating tasks, and enforcing business rules effectively.

## Topics Covered

* Introduction to PL/SQL
* PL/SQL Environment and Block Structure
* Variables and Data Types
* Basic PL/SQL Syntax
* Conditional Statements (IF, IF-ELSE, CASE)
* Looping Constructs (FOR, WHILE, LOOP)
* Exception Handling
* Implicit and Explicit Cursors
* Stored Procedures
* Functions
* Parameter Modes (IN, OUT, IN OUT)
* Packages and Package Bodies
* Database Triggers (BEFORE, AFTER, INSTEAD OF)

## Learning Outcomes

By completing this module, learners will be able to:

* Develop efficient database applications using PL/SQL.
* Implement robust exception handling mechanisms.
* Optimize data retrieval using cursors.
* Create and manage stored procedures and functions.
* Organize reusable code through packages.
* Automate database operations using triggers.
* Enforce business rules within database systems.
